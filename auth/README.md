# AUTH BOILERPLATE (CLIENT AND SERVER SIDE API)
## Using jwt and local strategy

## Server side:
0. navigate to /server directory
1. run 'npm install'
2. run 'mongod'
3. run 'npm run dev'

## Client side:
0. navigate to /client directory
1. npm install
2. run 'npm start
3. navigate to localhost:8080 in browser
